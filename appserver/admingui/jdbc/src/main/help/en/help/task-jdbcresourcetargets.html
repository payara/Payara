To Edit JDBC Resource Targets

  1. In the navigation tree, expand the Resources node.

  2. Expand the JDBC node.

  3. Expand the JDBC Resources node.

  4. Select the JDBC resource you wish to edit.

  5. From the Edit JDBC Resource page, select the Targets tab.

  6. To assign a new target to the resource, select the Manage Targets button.

  7. In the Manage Targets page, select an Available Target to Add it or remove it from the Selected Targets list.

  8. Click OK to apply your edits.
