passwords

securing and managing \{product---name}

Description

An administrator of \{product---name} manages one or more domains, each of which can have distinct administrative credentials. By managing a domain, an administrator effectively manages various resources like server instances, server clusters, libraries etc. that are required by the enterprise Java applications.
