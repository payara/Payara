To Set the Default Realm

The default realm is the realm that the GlassFish Server uses for authentication and authorization if an application’s deployment descriptor does not specify a realm.

  1. In the navigation tree, expand the Configuration node.
  2. Select the Security node.

    The Security page opens.

  3. From the Default Realm drop-down list, select the active realm that the server will use for authentication.

    The default value is file.

  4. Click Save.
