Resource Monitoring

Use the Resource Monitoring page to view monitoring data for a selected resource.

The Resource Monitoring page contains the following information.

Instance Name

The name of the server instance.

Connection Pool

The connection pool for which monitoring statistics are displayed.

For a description of the monitored properties, see Chapter 8, Administering the Monitoring Service, in Oracle GlassFish Server Administration Guide.
