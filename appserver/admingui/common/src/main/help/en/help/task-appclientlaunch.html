To Launch an Application Client

Before You Begin

Ensure that the application client you are launching is enabled. The client must be a web client that can be run with Java Web Start.

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. From the Deployed Applications table, click the name of the application client that you are launching.

    The name can include an optional version identifier, which follows the name and is separated from the name by a colon (:). For more information about module and application versions, see the Module and Application Versions in Oracle GlassFish Server Application Deployment Guide.

    The Edit Application page opens.

  3. Verify that the Java Web Start checkbox is selected.

    Java Web Start must be enabled before you can launch an application client. If the Java Web Start checkbox is not selected, select it, then click Save.

  4. In the Modules and Components table, click Launch under the Action column.

    The Launch Application Client page opens.

  5. On the Launch Application Client page, type the arguments to be passed to the application client.

  6. Click Launch.
