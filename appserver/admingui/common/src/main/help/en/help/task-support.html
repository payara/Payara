To View the GlassFish Server Blog

The GlassFish Server blog provides news from the GlassFish Server community and links to additional information about GlassFish Server.

  1. In the navigation tree, select the Common Tasks node.

    The Common Tasks page opens.

  2. Under GlassFish News, click the Blog link.

    The GlassFish Server blog opens in a separate browser tab or window.
