To View the Payara Server Blog

The Payara Server blog provides news from the Payara Server community and links to additional information about Payara Server.

  1. In the navigation tree, select the Common Tasks node.

    The Common Tasks page opens.

  2. Under Payara News, click the Blog link.

    The Payara Server blog opens in a separate browser tab or window.
