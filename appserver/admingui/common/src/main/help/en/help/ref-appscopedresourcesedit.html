To Modify Application Scoped Resources

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. Select an application for which you want to view the application-scoped resources.

    The application name can include an optional version identifier, which follows the name and is separated from the name by a colon (:).

    The page for the application opens.

  3. Select the Resources tab.

    The Application Scoped Resources page opens, and the resources currently specified for the application appear in a table with the following columns:

    Application Name

    The name of the application.

    Module Name

    The name of the module within the application.

    Resource Name

    The name of the application-scoped resource for the application or module.

    Resource Type

    The resource type, for example JDBC Connection Pools or Connector Resources.

  4. Click on the name of the resource you want to modify.

    The page that opens depends on the resource type. For example, if the resource is a JDBC connection pool, the Edit JDBC Connection Pool page opens.
