Add Resources

Use the Add Resources page to create one or more resources from a glassfish-resources.xml file on the specified target.

The Add Resources page contains the following information.

Location

The location of the glassfish-resources.xml file:

XML File to Be Uploaded to the Server

Path to the file on a remote machine.

Local XML File That Is Accessible from GlassFish Server

Path to the file on the domain administration server (DAS) machine.

Target

A drop-down list of clusters and stand-alone server instances.
