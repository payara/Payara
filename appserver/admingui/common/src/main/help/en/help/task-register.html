To View the Registration Page for GlassFish Server

The Registration page for GlassFish Server enables you to register your installation of GlassFish Server.

As a registered user of GlassFish Server, you receive benefits such as:

  • Notification of new versions, patches, and updates

  • Special offers on developer products, services, and training

  • Access to white papers, instructional articles, and live webinars with the GlassFish product team

  1. In the navigation tree, select the Common Tasks node.

    The Common Tasks page opens.

  2. Under GlassFish News, click the Registration link.

    The GlassFish Registration web page opens in a separate browser tab or window.
