JVM Profiler Settings

A profiler tool generates data that is used to analyze performance and identify potential bottlenecks.

Specify profiler settings. The information that you specify depends on which profiler product you are using. For examples and instructions, see Chapter 3, Debugging Applications, in Oracle GlassFish Server Application Development Guide.
