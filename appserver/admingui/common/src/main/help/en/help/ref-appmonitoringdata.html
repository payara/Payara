Application Monitoring

Use the Application Monitoring page to view monitoring data for a selected application.

The Application Monitoring page contains the following information.

Instance Name

The name of the server instance.

Application

The application for which monitoring statistics are displayed.

Component

The application component for which monitoring statistics are displayed.
