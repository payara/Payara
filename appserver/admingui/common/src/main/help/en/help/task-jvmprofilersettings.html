To Configure the JVM Profiler Settings

A profiler tool generates data that is used to analyze performance and identify potential bottlenecks.

  1. In the navigation tree, select the Configuration node.

  2. Select the configuration that you are changing.

  3. Under the Configuration node, select the JVM Settings node.

    The JVM General Settings page opens.

  4. On the JVM General Settings page, click the Profiler tab.

    The JVM Profiler Settings page opens.

  5. Specify profiler settings.

    The information that you specify depends on which profiler product you are using.

  6. Click Save.

  7. Restart the server.
