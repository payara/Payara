Configuration

Use the configuration node for a GlassFish Server instance or cluster to configure the following components:

  • JVM General Settings

  • Logger Settings

  • Web Container

  • EJB Container

  • Java Message Service

  • Security

  • Availability Service

  • Transaction Service

  • HTTP Service

  • Virtual Servers

  • Network Config

  • Thread Pools

  • ORB

  • Admin Service

  • Monitoring

  • Group Management Service

  • System Properties
