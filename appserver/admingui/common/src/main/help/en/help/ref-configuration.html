Configuration

Use the configuration node for a Payara Server instance or cluster to configure the following components:

  • JVM General Settings

  • Logger Settings

  • Web Container

  • EJB Container

  • Java Message Service

  • Security

  • Availability Service

  • Transaction Service

  • HTTP Service

  • Virtual Servers

  • Network Config

  • Thread Pools

  • ORB

  • Admin Service

  • Monitoring

  • System Properties
