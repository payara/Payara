To Collect and Download Domain Logs

You can collect all available log files for a specified GlassFish Server instance or cluster and compile the logs into a single ZIP archive.

Note:

If you specify a cluster, the ZIP archive also includes the log file for the domain administration server (DAS).

  1. In the navigation tree, select the Domain node.

    The Domain Attributes page opens.

  2. Select the Domain Logs tab.

    The Domain Logs page opens.

    If clusters exist, drop-down lists for clusters and instances are displayed. If no clusters exist, only a drop-down list for instances is displayed.

  3. Select either Cluster or Instance.

  4. Select the cluster or instance for which you are collecting and downloading log files.

  5. Click Collect Logs.

    Your browser's download mechanism prompts you to download the log-files-name.zip file, where name is the name of the cluster or instance.
