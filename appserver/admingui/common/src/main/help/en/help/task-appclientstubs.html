To Download Application Client Stubs

  1. In the navigation tree, select the Applications node.

    The Applications page opens.

  2. From the Deployed Applications table, click the name of the application client for which you are downloading stubs.

    The name can include an optional version identifier, which follows the name and is separated from the name by a colon (:).

    The Edit Application page opens.

  3. In the Modules and Components table, click Download Client Stubs under the Action column.

    The client-stubs.zip file is downloaded using your browser's download mechanism.
