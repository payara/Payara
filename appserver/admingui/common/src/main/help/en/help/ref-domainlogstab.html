Domain Logs

Use the Domain Logs page to collect and download log files for a specified GlassFish Server instance or cluster and compile the logs into a single ZIP archive.

Note:

If you specify a cluster, the ZIP archive also includes the log file for the domain administration server (DAS).

The Domain Logs page contains the following options.

Cluster

A drop-down list of clusters if any clusters exist.

Instance

A drop-down list of server instances.

Collect Logs

Button to collect the logs into a log-files-name.zip file, where name is the name of the cluster or instance.

The log-files-name.zip file is downloaded using your browser's download mechanism.
