To Create an IIOP Listener

  1. In the tree component, expand the Configuration node.

  2. Select the instance to configure:

    • To configure a particular instance, select the instance's config node. For example, for the default instance, server, select the server-config node.

    • To configure the default settings for future instances that use a copy of default-config, select the default-config node.

  3. Under the Configuration node, expand the ORB node.

  4. Select IIOP Listeners.

    The IIOP Listeners page opens.

  5. Click New.

  6. In the Name field, type a name to identify the listener.

  7. In the Network Address field, type the network address of the listener.

    You can type an IP address or a DNS resolvable host name.

  8. In the Listener Port field, type the port number upon which the IIOP listener is to listen.

  9. Select the Listener Enabled checkbox to enable ORB IIOP inbound connections to the GlassFish Server.

    This option is enabled by default.

  10. Select the Security Enabled checkbox to enable SSL for the IIOP listener.

    This option is disabled by default.

  11. In the Additional Properties section, specify additional properties.

    The GlassFish Serverdoes not define any additional properties for IIOP listeners.

    To add a property, click the Add Property button. In the blank row that appears, type the property name in the Name field, and type the property value in the Value field.

  12. Click OK.

  13. If you selected the Security Enabled checkbox, follow the steps in To Edit SSL Settings for a Secure IIOP Listener.
