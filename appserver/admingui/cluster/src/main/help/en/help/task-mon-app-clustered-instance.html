To View Application Monitoring Data for a Clustered GlassFish Server Instance

Before You Begin

Before you can view monitoring data, you must configure the monitoring service.

  1. In the navigation tree, expand the Clusters node.

  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Instances tab.

    The Clustered Server Instances page for the cluster opens.

  4. Select the instance for which to view application monitoring data.

    The General Information page for the instance opens.

  5. On the General Information page, click the Monitor tab.

    The Application Monitoring page opens.

  6. From the View drop-down list, select the application for which you want to view monitoring data.

  7. To update displayed monitoring data, click Refresh.
