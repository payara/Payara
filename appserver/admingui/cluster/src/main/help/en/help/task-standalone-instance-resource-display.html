To List Resources for a Standalone GlassFish Server Instance

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. On the Standalone Server Instances page, select the instance.

    The General Information page for the instance opens.

  3. On the General Information page, click the Resources tab.

    The Resources page for the instance opens.

  4. To display only resources of a particular type, from the Filter pull-down list, select the type of resource to display.
