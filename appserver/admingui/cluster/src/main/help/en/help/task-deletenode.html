To Delete Nodes

Deleting a node removes the node from the configuration of the domain administration server (DAS). The node's directories and files are deleted when the last GlassFish Server instance that resides on the node is deleted.

You can also choose to uninstall GlassFish Server software from the node host when the node is deleted.

Before You Begin

Ensure that no GlassFish Server instances reside on the node that is being deleted.

  1. In the navigation tree, select the Nodes node.

    The Nodes page opens.

  2. For each node that you are deleting, select the check box adjacent to the node in the Nodes table.

    Note:

    The predefined node localhost-domain cannot be deleted.

  3. Click the button to delete the nodes or delete the nodes and uninstall GlassFish Server software from the node hosts.

    • To delete the nodes without uninstalling the GlassFish Server software, click Delete.

      You are asked to confirm that you want to delete the nodes.

    • To delete the nodes and uninstall GlassFish Server software from the node hosts, click Delete and Uninstall.

      You are asked to confirm that you want to delete the nodes and uninstall GlassFish Server software from the node hosts.

      Note:

      The GlassFish Server software is not uninstalled from any host that a node of type CONFIG represents. Only the node is deleted.

  4. Confirm that you want to delete the nodes or delete the nodes and uninstall GlassFish Server software from the node hosts.

    The nodes are deleted and are removed from the table of nodes on the Nodes page. If you clicked Delete and Uninstall, the GlassFish Server software is also uninstalled from the node hosts.
