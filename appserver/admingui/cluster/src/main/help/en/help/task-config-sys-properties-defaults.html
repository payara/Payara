To Set Default Values for Configuration System Properties

The default values of configuration properties are defined in the configuration that the cluster or instance references.

  1. In the navigation tree, expand the node for the configuration.

  2. From the list of property types under the expanded node, select System Properties.

    The System Properties page for the selected configuration opens.

    This page displays the current values of the following system properties for the configuration:

    • Port numbers that are predefined when the configuration was created

    • Custom properties that are added to the configuration, if any

  3. To ensure that your changes are applied without the need to restart cluster or instance, select the Dynamic Reconfiguration option.

  4. For each property for which you are setting a default value, type the new value in the Default Value field for the property.

  5. Set default values for any additional properties that you require.

    1. In the Additional Properties table, click Add Property.

      A row is added to the Additional Properties table.

    2. In the Instance Variable Name field, type the property name.

    3. In the Default Value field, type the value.

  6. Click Save.
