To Edit an Application That Is Deployed to a Standalone GlassFish Server Instance

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. On the Standalone Server Instances page, select the instance.

    The General Information page for the instance opens.

  3. On the General Information page for the instance, click the Applications tab.

    The Applications page for the instance opens.

  4. In the Deployed Applications table on the Applications page, click the application name.

    The Edit Application page opens.

  5. On the Edit Application page, provide the required information about the application and save your changes.

    For details, see the Edit Application online help.
