To Undeploy Applications From a Cluster

Undeploying an application removes the application from all targets and the domain. When an application is undeployed, the application is no longer listed on the Applications page that displays all applications that are deployed in GlassFish Server.

  1. In the navigation tree, expand the Clusters node.

  2. Under the expanded Clusters node, select the node for the cluster.

    The General Information page for the cluster opens.

  3. On the General Information page, click the Applications tab.

    The Applications page for the cluster opens.

  4. For each application that you are undeploying, select the check box adjacent to the application in the Deployed Applications table.

  5. Click Undeploy.

    You are asked to confirm that you want to undeploy the applications.

  6. Confirm that you want to undeploy the applications.

    The applications are undeployed and are removed from the table of deployed applications on the page.
