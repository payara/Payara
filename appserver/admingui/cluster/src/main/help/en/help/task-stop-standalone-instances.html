To Stop Standalone GlassFish Server Instances

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. For each instance that you are stopping, select the check box adjacent to the instance in the Server Instances table.

  3. Click Stop.

    You are asked to confirm that you want to stop the instances.

  4. Confirm that you want to stop the instances.

    The status of each instance is updated in the Server Instances table when the instance is stopped.
