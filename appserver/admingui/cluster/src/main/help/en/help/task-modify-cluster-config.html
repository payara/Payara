To Modify the Configuration of a Cluster

  1. In the navigation tree, expand the Clusters node.

  2. Select the node for the cluster.

  3. Click the name of the configuration.

    The Configuration page for the cluster opens.

  4. On the Configuration page, select the component of the configuration to modify.

    The page for editing the selected component of the configuration opens.

  5. On the page for editing the selected component, make the required changes and save your changes.

    For details, see the online help for the page for editing the selected component.
