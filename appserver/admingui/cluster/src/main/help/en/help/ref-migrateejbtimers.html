Migrate EJB Timers

The Migrate EJB Timers page contains the following options.

Source

Drop-down list of stopped GlassFish Server instances in the cluster. EJB timers can be migrated only from instances that are not running.

Destination

Drop-down list of running GlassFish Server instances in the cluster. EJB timers can be migrated only to instances that are running.

Note:

If all instances in the cluster are not running, this page displays only a warning that no running instances to which EJB timers can be migrated are available.

The Migrate EJB Timers page contains a table of EJB timers with the following information.

Server Name

List of GlassFish Server instances in the cluster.

Timer Count

Number of persistent timers on each instance.
