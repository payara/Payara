To Delete Standalone GlassFish Server Instances

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. For each instance that you are deleting, select the check box adjacent to the instance in the Server Instances table.

  3. Click Delete.

    You are asked to confirm that you want to delete the instances.

  4. Confirm that you want to delete the instances.

    The instances are deleted and are removed from the table of instances on the page.
