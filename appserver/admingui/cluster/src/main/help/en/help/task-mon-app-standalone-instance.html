To View Application Monitoring Data for a Standalone GlassFish Server Instance

Before You Begin

Before you can view monitoring data, you must configure the monitoring service.

  1. In the navigation tree, select the Standalone Instances node.

    The Standalone Server Instances page opens.

  2. Select the instance for which to view application monitoring data.

    The General Information page for the instance opens.

  3. On the General Information page, click the Monitor tab.

    The Application Monitoring page opens.

  4. From the View drop-down list, select the application for which you want to view monitoring data.

  5. To update displayed monitoring data, click Refresh.
