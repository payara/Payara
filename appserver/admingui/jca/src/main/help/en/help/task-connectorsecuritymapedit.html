To Edit a Security Map for a Connector Connection Pool

  1. In the navigation tree, expand the Resources node.

  2. Expand the Connectors node.

  3. Under the Connectors node, select the Connector Connection Pools node.

    The Connector Connection Pools page opens.

  4. On the Connector Connection Pools page, click the name of the connection pool that you are editing.

    The Edit Connector Connection Pool page opens.

  5. On the Edit Connector Connection Pool page, click the Security Maps tab.

    The Connector Connection Pool Security Maps page opens.

  6. On the Connector Connection Pool Security Maps page, click the name of the security map you are editing.

    The Edit Connector Connection Pool Security Map page opens.

  7. Do one of the following, but not both:

    • Select the User Groups radio button and type the names of the user groups, separated by commas.

    • Select the Principals radio button and type the names of the principal, separated by commas.

  8. In the Username field, specify the backend principal username required by the EIS.

  9. In the Password field, specify the backend principal password required by the EIS.

  10. Click Save.
