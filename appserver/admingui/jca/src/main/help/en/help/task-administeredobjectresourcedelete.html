To Delete an Administered Object Resource

  1. In the navigation tree, expand the Resources node.

  2. Expand the Connectors node

  3. Under the Connectors node, select the Admin Object Resources node.

    The Admin Object Resources page opens.

  4. On the Admin Object Resources page, select the checkbox next to the name of the administered object resource that you are deleting.

  5. Click Delete.
