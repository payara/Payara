To View Statistics for a JMS Physical Destination

  1. In the navigation tree, expand the Configuration node.

  2. Under the Configuration node, expand the Java Message Service node.

  3. Under the Java Message Service node, select the Physical Destinations node.

    The Physical Destinations page opens.

  4. On the Physical Destinations page, click the View link for the destination for which you are viewing statistics.

    The View JMS Physical Destination Statistics page opens.

  5. When you have finished viewing the statistics, click the Back button.
