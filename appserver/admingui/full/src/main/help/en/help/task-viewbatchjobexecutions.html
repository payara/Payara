To View Batch Job Executions

  1. In the navigation tree, select the GlassFish Server cluster or server instance for which you want to view batch job executions.

    The General Information page opens.

  2. Click the Batch tab.

    The Batch Job Executions page opens.

  3. On the Batch Job Executions page, click an execution ID to view details about that execution.

    The Batch Job Execution Details page opens.

  4. To view details about steps in the batch job execution, click the Execution Steps tab.

    The Batch Job Execution Steps page opens.

  5. Click the Back button to return to the Batch Job Executions page.
